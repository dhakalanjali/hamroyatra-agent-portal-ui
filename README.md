# Hamroyatra - Agent Portal Dashboard UI

A hotel and booking admin dashboard UI designed in Figma for my second year college project.

## About

Hamroyatra is an agent portal where hotel agents can manage their bookings, revenue, listings, and customers from one place.

## Features

- Overview of total bookings, revenue and active listings
- Operational revenue chart
- Activity ledger with booking status
- Sidebar with Calendar, Bookings, Listings, Messages, Teams, Customers

## Tool Used

- Figma
